## File Structure
```
riskan-portfolio/
├── index.html      ← Main HTML page
├── style.css       ← All styles
├── script.js       ← Cursor, animations, typing effect
└── README.md       ← This file
```

## How to Deploy on GitHub Pages

1. Create a new repo on GitHub named **riskan-portfolio**
2. Upload all 3 files (index.html, style.css, script.js)
3. Go to **Settings → Pages**
4. Under **Source**, select **main** branch → **/ (root)**
5. Click **Save**
6. Your site will be live at: `chipper-maamoul-f84522.netlify.app`
